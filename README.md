# GNOME Meme Generator

The ultimate tool for creating memes built upon GTK-4 and Libadwaita for the GNOME desktop.

## Features
- Import images
- Add top and bottom text
- Export memes as PNG
- Clean, native GNOME interface
- Let It Happen
